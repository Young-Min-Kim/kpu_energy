<h2>무엇을 위한 시스템인가요? </h2>
본 시스템은 매장 내 전자기기의 전력사용량을 모니터링하며,<br>카메라를 통한 모션인식/얼굴인식을 통해 매장 손님수를 파악하고,
<br>손님이 없을 경우 전자기기의 전원을 차단 및 출력을 조정하여 에너지 절약을 실현합니다
